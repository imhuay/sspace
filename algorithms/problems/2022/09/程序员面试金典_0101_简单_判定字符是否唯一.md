<summary><b>思路：排序</b></summary>

- 因为要求不使用额外数据结构，因此可以考虑排序后，顺序两两比较；
- 注意边界条件；

<details><summary><b>Python</b></summary>

```python
class Solution:
    def isUnique(self, astr: str) -> bool:
        if not astr: return True
        
        cs = sorted(astr)
        pre = cs[0]
        for c in cs[1:]:
            if pre == c:
                return False
            pre = c

        return True
```

</details>
