```txt
写出一个程序, 接受一个字符串, 然后输出该字符串反转后的字符串. (字符串长度不超过1000)
```

<summary><b>思路: 双指针</b></summary>

- 设置首尾双指针, 交换字符, 直到相遇;

<details><summary><b>Python</b></summary>

```python
class Solution:
    def solve(self , s: str) -> str:

        s = list(s)
        l, r = 0, len(s) - 1
        while l < r:
            s[l], s[r] = s[r], s[l]
            l += 1
            r -= 1

        return ''.join(s)
```

</details>
