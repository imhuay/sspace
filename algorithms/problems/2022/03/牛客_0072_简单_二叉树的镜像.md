<summary><b>思路</b></summary>

- 后序遍历，交换左右子节点；

<details><summary><b>Python</b></summary>

```python
class Solution:
    def Mirror(self , pRoot: TreeNode) -> TreeNode:
        
        def dfs(x):
            if not x: return None
            x.right, x.left = dfs(x.left), dfs(x.right)
            return x
        
        return dfs(pRoot)
```
