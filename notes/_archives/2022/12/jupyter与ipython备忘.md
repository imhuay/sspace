## 安装
```shell
pip install jupyterlab
# pip install notebook
```
将同时安装 notebook, ipython 等一系列模块;

## 启动
```shell
## 默认参数启动
jupyter lab

## 常用参数
jupyter lab --port=9999 --no-browser --autoreload --ip=xxx
```
